## Setup and Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/0x3mr/RemyRecipeX
   cd RemyRecipeX
   ```
2. Set up python3-tk:
   - `sudo apt install python3-tk`

3. Install requirments:
   - `pip3 install -r requirments.txt`

4. Run
   - `python3 main.py`
